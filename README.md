## Project Description

BirthdayDrop is a fun and interactive web application designed to celebrate birthdays. It allows users to create a personalized birthday page with a countdown to their special day, a section for friends and family to leave wishes, a virtual cake with candles to blow out, and a surprise wheel for extra fun. The application is built using HTML, CSS, and JavaScript, focusing on a delightful user experience with engaging animations and a festive design.

## Features

*   **Interactive Intro Screen**: A welcoming screen that transitions into the main application with a confetti burst.
*   **Personalized Birthday Countdown**: Displays the remaining time until the user's birthday.
*   **Special Message**: A customizable message for the birthday person.
*   **Virtual Birthday Cake**: An interactive cake with candles that can be 
blown out, triggering fireworks.
*   **Surprise Wheel**: A spinning wheel that reveals fun birthday messages.
*   **Memories Section**: A section to display heartwarming birthday memories or messages.
*   **Wish Book**: A dynamic section where visitors can add their birthday wishes, stored locally in the browser.
*   **Shareable Link**: Generate a unique link to share the personalized birthday page.
*   **Confetti and Fireworks**: Engaging visual effects to enhance the celebratory atmosphere.

## Technologies Used

*   **HTML5**: For structuring the web content.
*   **CSS3**: For styling and animations, creating a visually appealing and responsive design.
*   **JavaScript (ES6+)**: For interactive functionalities, dynamic content updates, and event handling.
*   **Canvas Confetti**: A lightweight JavaScript library for confetti animations.

## Setup and Installation

To set up the project locally, follow these steps:

1.  **Clone the repository**:

    ```bash
    git clone https://github.com/YOUR_USERNAME/BirthdayDrop.git
    ```

2.  **Navigate to the project directory**:

    ```bash
    cd BirthdayDrop
    ```

3.  **Open `index.html`**: Simply open the `index.html` file in your web browser to run the application.

## Usage

1.  **Enter Your Name and Birth Date**: On the main page, enter the birthday person's name and birth date.
2.  **Start the Magic**: Click the "انقر هنا لرؤية السحر 🚀" button to activate the countdown, special message, and other interactive elements.
3.  **Share the Link**: Copy the generated shareable link and send it to friends and family.
4.  **Interact**: Blow out the candles on the cake, spin the surprise wheel, and read/add wishes.
